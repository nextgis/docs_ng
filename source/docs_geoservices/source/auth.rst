Authorization
============

To log in to `NextGIS Geoservices <https://geoservices.nextgis.com/>`_ you must use a `NextGIS ID <https://my.nextgis.com>`_ account (:numref:`nggeos_auth`).
If you don't have an account you need to sign up.
