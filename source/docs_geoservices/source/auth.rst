Авторизация
============

Для авторизации на  `NextGIS Geoservices <https://my.nextgis.com>`_ используется учетная запись `NextGIS ID <https://my.nextgis.com>`_ (:numref:`nggeos_auth`).
При отсутствии таковой необходимо пройти регистрацию.
