Документация по NextGIS Лес
============================

Веб-приложение NextGIS Лес позволяет вести учет лесосек и объектов инфраструктуры, подготавливать лесную декларацию и отчеты об использовании/воспроизводстве лесов, осуществлять мониторинг пожарной обстановки. 

Сервис ориентирован на:

* Лесозаготовителей 
* Компании, чья деятельность подразумевает попутную вырубку лесных насаждений
* Компании, занимающиеся таксацией и отводом лесосек и подготовкой отчетности

Документация по веб-приложению разделена на два блока. 

**Руководство для администратора** предназначено для ГИС-специалиста, который будет заниматься первоначальным наполнением и настройкой приложения и, по мере необходимости, обновлением данных (космосъемка, материалы лесоустройства).

**Руководство для пользователя** предназначено для сотрудника с любым уровнем компетенций, чьи задачи – внести информацию об отведенной лесосеке/объекте инфраструктуры, сформировать лесную декларацию или отчет, отследить пожарную обстановку на подконтрольном участке.
